1. 처음이라면 `02-common-antipatterns.md`부터 읽으세요
2. 회고할 때는 `01-retrospective-frameworks.md`의 템플릿을 활용하세요
3. AI에게 요청이 잘 안 될 때 `03-prompt-engineering.md`를 참고하세요
4. 모르는 개발 용어가 나오면 `04-development-concepts.md`를 찾아보세요
